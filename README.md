# WLED Glow

A simple, beautiful Progressive Web App to control your WLED LED lights.

**Live Demo:** [https://jcrigby.github.io/wled/](https://jcrigby.github.io/wled/)

## Features

- **One-tap power control** - Large, easy-to-use ON/OFF button
- **Brightness slider** - Smooth, real-time brightness adjustment
- **Quick colors** - Preset color swatches for instant mood changes
- **80+ effects** - Browse by category (Relaxing, Energetic, Nature, Party)
- **Favorites** - Save and recall your favorite lighting setups
- **Demo mode** - Try the full app without any hardware
- **Works offline** - Install as a PWA on your phone or tablet

## Getting Started

1. Visit the [live demo](https://jcrigby.github.io/wled/)
2. The app starts in Demo Mode so you can explore all features
3. To connect real lights, go to Settings and enter your WLED device's IP address

## About WLED

[WLED](https://kno.wled.ge/) is open-source software that runs on ESP32/ESP8266 microcontrollers to control addressable LED strips. WLED Glow provides a simplified interface for everyday use.

## Tech Stack

- Vanilla JavaScript (no framework dependencies)
- CSS with custom properties for theming
- Service Worker for PWA/offline support
- WLED JSON API for device communication
